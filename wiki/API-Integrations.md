# API Integrations

## Cloudinary

- **Purpose**: Image storage and CDN
- **Features**: Automatic optimization, responsive images
- **Usage**: Farm and product image uploads

## OpenStreetMap (flutter_map)

- **Purpose**: Interactive maps for farm discovery
- **Features**: Farm markers, user location, distance calculation
- **Advantage**: Free, no API key required

## OpenWeatherMap (Planned)

- **Purpose**: Weather forecasts for harvest planning
- **Features**: 7-day forecast, weather alerts
- **Status**: Backend ready, UI integration planned

## Firebase Services

| Service | Usage |
|---------|-------|
| **Firestore** | Primary database, offline-first |
| **Auth** | Phone OTP authentication |
| **FCM** | Push notifications |
| **Analytics** | User behavior tracking |
| **Hosting** | Web app deployment |
